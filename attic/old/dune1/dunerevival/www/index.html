  SUMMARY

 

This project's goal is to program a kind of Virtual Machine to allow people who already own legally the game "Dune" (all rights reserved to Electronics Arts) to play it on modern machines, with all the comfort provided by modern O.S. (windowed mode, sound level control, antialiasing, etc.)

 

  
 

 
 

PROJECT DETAILS

The project will be conducted in two steps :

1/ Developement of a viewer to extract data from original files : HSQ files, AGD files, etc.

2/ Developement of the Virtual Machine to play the game.
