# Dune 2 Script Decompiler / Compiler

#### About

DST is an EMC script compiler/decompiler for Dune 2, this grants you the possibility of modifying the in-game scripts.

#### Usage

##### Decompile
dst d build.emc  
dst d team.emc  
dst d unit.emc  

##### Compile
dst c build.txt  
dst c team.txt  
dst c unit.txt
